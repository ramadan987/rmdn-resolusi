TABLE TAG
1. **`table`**: Membungkus seluruh elemen tabel. 2. **`caption`**: Menambahkan deskripsi atau judul untuk tabel (opsional). 3. **`thead>`**: Grup untuk baris header, biasanya mencakup elemen `th`. 4. **`tbody`**: Bagian utama tabel, berisi isi tabel dengan elemen `td`. 5. **`tfoot`**: Grup untuk baris footer di bagian bawah tabel. 6. **`tr`**: Digunakan untuk membuat baris dalam tabel. 7. **`th`**: Sel header tabel, biasanya digunakan di dalam `thead`. 8. **`td`**: Sel isi tabel, digunakan di dalam `tbody` atau `tfoot`.

colgroup untuk memberikan style/warna pada setiap kolom jika ingin menggunakan style pada satu kolom maka gunakkan span=

colspan = untuk menggabungkan beberapa kolom di baris yang sama rowspan = menggabungkan beberpa kolom di baris yang berbeda
