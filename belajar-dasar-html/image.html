belajar image

image attribut
attribute src = yang di gunakan untuk menentukan lokasi gambar yang mau di tampilkan
bisa menggunakan absolute URL atau relatif URL
attribute alt = di gunakan untuk text gambar

image siize
width untuk mengubah ukuran lebar
weight untuk mengubah posisi tinggi
kita bisa menggunakan uuran px (pixel) untuk perssenntase dari ukuran asli
